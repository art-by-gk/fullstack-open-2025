sequenceDiagram
    participant browser
    participant server

    browser->>browser: Add the new note to the local notes list
    browser->>browser: Update the notes shown on the page

    activate server
    Note over server: Server stores the new note
    server-->>browser: 201 Created (successful response)
    deactivate server
